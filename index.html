What is free jqGrid?

Free jqGrid allows to create comfortable and performant grids having many additional features like sorting, grouping and so on. It's based on jqGrid, which was provided for free under MIT and GPL v2 licenses up to version 4.7. The license agreement was changed in jqGrid 4.7.1. One can say that starting with the version 4.7.1 there are no more jqGrid, but there exist commertial product Guriddo jqGrid JS instead. Free jqGrid will be developed based on jqGrid 4.7 and provided under MIT and GPL v2 licenses. One can use it without and mandotory fee. We just ask to contribute the development by donating via PayPal, if one have the possibility for it.

What is the goal of the documentation?

The current site is the first attempt to create new documentation of jqGrid with description of all new features of free jqGrid and to share the set of examples of usage free jqGrid. The information will be extended and probably the look of the page will be changed too. I just decided to to post *the early preliminiry* version to show other developers, who uses jqGrid, in which direction I work now. I ask all developers to share his expirience. I suggest to post "feature request to documentation" in the Issues of the repository. One can request, for example, for the demo which shows the usage free jqGrid in the corresponding test case.

The look of the web site can be changed soon. The main goals of the next changes is providing the way in which the content will be extended without breaking the URLs to the previously posted content. Additional goal is separation the list of options, functions and other long collection of infornation from HTML code. We don't want to use any CMS currently. It should still be possile to use for example the URL to options with sufix ?filter=prmNames or #prmNames to filter or to jump to the specified option. The requirement exist at least for the pages which displays methods or options. It should be possible to filter the information to some specific category. For example to display the options, methods, callbacks or event, which are related to TreeGrid without displaying for example the information related for data grouping. We want to make possible to display long information not only table form, but using <li> for example.

We ask all native English speakers and all, who can improve the text of the web site, to post his/her suggesting as pull requests to free-jqgrid.github.io
